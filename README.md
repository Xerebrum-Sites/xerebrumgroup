# xerebrumgroup

Sitio web de la consultora Xerebrum.

## Stack

- React 18
- Vite
- TypeScript
- Tailwind CSS
- Cloudflare Pages

## Desarrollo local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Se deploya automáticamente en Cloudflare Pages desde la rama `main`.
